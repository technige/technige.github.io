# <img src="gu/firebrand.profession.png" style="display:inline-block;height:100%;vertical-align:middle"> <span style="display:inline-block;height:100%;vertical-align:middle">Firebrand</span>

**Firebrand** is an elite specialization for the Guardian.

The table below lists the full set of core specialization combinations available for the Firebrand.
Seven of these provide synergy with an off-hand weapon and therefore can be effectively paired with the Firebrand's axe.

| Core Specializations  | Weapon Synergies                                                                     |
|-----------------------|--------------------------------------------------------------------------------------|
| **Zeal, Radiance**    | Greatsword, Scepter+Torch, Sword+Torch, **Axe**+Torch                                |
| **Zeal, Valor**       | Greatsword, Scepter+Focus, Scepter+Shield, **Axe**+Focus, **Axe**+Shield             |
| Zeal, Honor           | Greatsword, Staff                                                                    |
| Zeal, Virtues         | Greatsword, Hammer                                                                   |
| **Radiance, Valor**   | Sword+Torch, Sword+Focus, Sword+Shield, **Axe**+Torch, **Axe**+Focus, **Axe**+Shield |
| **Radiance, Honor**   | Sword+Torch, Mace+Torch, **Axe**+Torch, Staff                                        |
| **Radiance, Virtues** | Sword+Torch, **Axe**+Torch, Hammer                                                   |
| **Valor, Honor**      | Mace+Focus, Mace+Shield, **Axe**+Focus, **Axe**+Shield, Staff                        |
| **Valor, Virtues**    | **Axe**+Focus, **Axe**+Shield, Hammer                                                |
| Honor, Virtues        | Staff, Hammer                                                                        |
